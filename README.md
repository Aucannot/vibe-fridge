## vibe-fridge

### 🚀 快速开始

#### 1. 克隆代码仓库

```bash
git clone YOUR_GITHUB_REPO_URL
cd vibe-fridge
```

#### 2. 创建并激活虚拟环境（推荐）

macOS / Linux：

```bash
python3 -m venv .venv
source .venv/bin/activate
```

Windows（PowerShell）：

```bash
python -m venv .venv
.venv\Scripts\activate
```

#### 3. 安装依赖

```bash
pip install -r requirements.txt
```

#### 4. 启动应用

```bash
python -m app.main
```

或直接：

```bash
python app/main.py
```

启动后会打开一个 360×640 的 KivyMD 窗口，它就是 vibe-fridge 的主界面。
